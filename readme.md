# Doomsday Quiz

A simple web game to practice the Doomsday Algorithm.

Given a random date, guess the correct day of the week within a time limit.

---
## Demo

[Play the Game](https://doomsday-quiz.vercel.app/)

---

## Features

- 60-second timer
- Random date generation
- Instant feedback on answers
- Hint system (hover to see anchor day)
- Visual urgency (color + animation when time is low)

---

## How It Works

The Doomsday Algorithm lets you find the day of the week by using a few reference dates. 

Each year has an “anchor day,” and certain dates (e.g., 4/4, 6/6, 8/8) always fall on that same weekday.

---

## How to Play

1. Click **START**
2. Look at the given date
3. Choose the correct day (Sun–Sat)
4. Score as many points as possible before time runs out
